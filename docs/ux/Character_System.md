# Character System - Персонажи-наставники

> **Назначение:** Система персонажей-наставников для персонализированного обучения
> **Phase:** Phase 1 (текстовые подсказки) → Phase 2+ (визуал + голос)
> **Для кого:** Все пользователи (выбор персонажа при онбординге)
> **Версия:** 1.0
> **Дата:** 16 ноября 2025

---

## Обзор

**Character System** — это система персонажей-наставников, которые сопровождают пользователя на всем пути обучения слепой печати. Каждый персонаж имеет уникальный стиль общения, подходящий для своей целевой аудитории.

### Зачем нужны персонажи?

1. **Персонализация** — пользователь выбирает наставника под себя
2. **Вовлечение** — дружелюбный персонаж мотивирует продолжать
3. **Эмоциональная связь** — обучение становится интереснее
4. **Геймификация** — для детей это полноценный игровой опыт
5. **Снижение стресса** — поддержка при ошибках, поощрение при успехах

### Принцип подбора персонажа

| Пользователь | Персонаж | Логика |
|--------------|----------|--------|
| Мужчина (взрослый) | Анна 👩‍🏫 | Женский наставник для мужчин |
| Женщина (взрослая) | Максим 👨‍💼 | Мужской наставник для женщин |
| Ребенок | Кнопыч 🤖 или Клавочка 🎨 | Выбор ребенка (игровые персонажи) |

**Психология:** Исследования показывают, что люди лучше воспринимают инструкции от противоположного пола. Для детей — важна визуальная привлекательность и игровой стиль.

## Персонаж 1: Анна 👩‍🏫

### Профиль

```
┌─────────────────────────────────────────┐
│             👩‍🏫 Анна                    │
├─────────────────────────────────────────┤
│  Роль:      Учительница печати          │
│  Для:       Мужчины (взрослые)          │
│  Возраст:   ~30 лет                     │
│  Стиль:     Поддерживающий, дружелюбный │
│  Tone:      Ты-форма, неформальный      │
└─────────────────────────────────────────┘
```

### Характер

- **Дружелюбная** — обращается по имени, создает атмосферу доверия
- **Поддерживающая** — всегда найдет что-то позитивное, даже при ошибках
- **Мотивирующая** — подчеркивает прогресс, вдохновляет продолжать
- **Терпеливая** — никогда не ругает за ошибки
- **Профессиональная** — дает четкие советы, но без занудства

### Tone of Voice

- **Обращение:** "Ты" (неформальное)
- **Стиль:** Дружелюбный, но профессиональный
- **Эмодзи:** Умеренно (1-2 за сообщение): 💪 🎯 ✨ ⭐ 🚀
- **Длина фраз:** Короткие предложения (5-12 слов)
- **Никогда не использует:** Сленг, жаргон, "детские" выражения

### Примеры фраз

**При начале урока:**
- "Привет, {имя}! Готов к новому уроку? 💪"
- "{имя}, сегодня изучаем средний палец! Будет интересно! 🎯"
- "Начинаем, {имя}! Помни: точность важнее скорости ✨"

**Во время урока (хорошо):**
- "Отлично, {имя}! Уже {wpm} WPM! 🚀"
- "Вау! Точность {accuracy}%! Продолжай в том же духе!"
- "Ты на правильном пути, {имя}! ⭐"

**Во время урока (ошибки):**
- "Эй, не спеши! Лучше медленно, но правильно 🎯"
- "{имя}, притормози немного. Точность важнее!"
- "Ничего страшного, {имя}. Попробуй еще раз 💪"

**Урок завершен (успех):**
- "Красава, {имя}! ✅ Ошибки: {errors} из {limit} — отличный результат!"
- "Поздравляю! {wpm} WPM и {accuracy}% точности! 🎉"
- "{имя}, ты молодец! Переходим к следующему? 🚀"

**Превышен лимит ошибок:**
- "Упс, {имя}! Ошибок многовато ({errors} из {limit}). Попробуем еще раз? 💪"
- "Не переживай, {имя}. Давай еще раз, более внимательно 🎯"
- "{имя}, почти получилось! Сосредоточься и повтори ✨"

**Новый уровень разблокирован:**
- "ПОЗДРАВЛЯЮ, {имя}! 🎉 Уровень '{level}' открыт!"
- "{имя}, ты справился! Новый уровень ждет тебя! 🚀"
- "Отлично, {имя}! Готов к новым вызовам? ⭐"

**Weak Keys (частые ошибки):**
- "{имя}, замечаю, что буква '{key}' дается сложно. Хочешь дополнительную тренировку?"
- "Видимо, клавиша '{key}' — твое слабое место. Давай потренируем! 💪"

## Персонаж 2: Максим 👨‍💼

### Профиль

```
┌─────────────────────────────────────────┐
│            👨‍💼 Максим                   │
├─────────────────────────────────────────┤
│  Роль:      Опытный наставник           │
│  Для:       Женщины (взрослые)          │
│  Возраст:   ~35 лет                     │
│  Стиль:     Профессиональный, с юмором  │
│  Tone:      Ты-форма, но чуть строже    │
└─────────────────────────────────────────┘
```

### Характер

- **Профессиональный** — серьезно относится к обучению
- **Опытный** — дает конкретные советы, основанные на "опыте"
- **С юмором** — иногда шутит, но не переходит грань
- **Требовательный** — подчеркивает важность точности
- **Справедливый** — хвалит за достижения, но указывает на ошибки

### Tone of Voice

- **Обращение:** "Ты" (неформальное, но уважительное)
- **Стиль:** Профессиональный с легкой иронией
- **Эмодзи:** Редко (1 за сообщение): 🎯 ✅ 📈 💼 ⚡
- **Длина фраз:** Средние предложения (7-15 слов)
- **Использует:** "Отлично", "Превосходно", "Точно", "Верно"

### Примеры фраз

**При начале урока:**
- "{имя}, приветствую! Сегодня — уровень {level}. Готова?"
- "Начинаем, {имя}. Сосредоточься на технике 🎯"
- "{имя}, новый урок — новые возможности. Поехали!"

**Во время урока (хорошо):**
- "Превосходно, {имя}! {wpm} WPM — отличный темп! ⚡"
- "{имя}, точность {accuracy}% — так держать! ✅"
- "Именно так, {имя}! Продолжай в том же духе 📈"

**Во время урока (ошибки):**
- "{имя}, слишком торопишься. Притормози 🎯"
- "Внимание, {имя}! Ошибки растут. Сосредоточься"
- "{имя}, помни: качество > количество"

**Урок завершен (успех):**
- "Отлично, {имя}! ✅ {wpm} WPM и всего {errors} ошибок!"
- "{имя}, превосходный результат! {accuracy}% точности! 🎯"
- "Так держать, {имя}! Следующий уровень будет сложнее, но ты справишься ⚡"

**Превышен лимит ошибок:**
- "{имя}, ошибок слишком много: {errors} из {limit}. Повтори внимательнее"
- "Не получилось, {имя}. Но это нормально — повторение мать учения 💼"
- "{имя}, сосредоточься и попробуй еще раз. У тебя получится 🎯"

**Новый уровень разблокирован:**
- "Поздравляю, {имя}! Уровень '{level}' разблокирован! ✅"
- "{имя}, отличная работа! Новые вызовы ждут 📈"
- "Молодец, {имя}! Переходим на следующий уровень ⚡"

**Weak Keys:**
- "{имя}, клавиша '{key}' требует внимания. Дополнительная практика?"
- "Замечаю трудности с '{key}', {имя}. Предлагаю усиленную тренировку 🎯"

## Персонаж 3: Кнопыч 🤖

```
┌─────────────────────────────────────────┐
│            🤖 Кнопыч                     │
├─────────────────────────────────────────┤
│  Роль:      Робот-клавиша               │
```

### Характер

- **Веселый** — всегда в хорошем настроении
- **Энергичный** — использует восклицания, эмодзи
- **Игривый** — превращает обучение в игру
- **Поощряющий** — награды, значки, достижения
- **Современный** — использует игровую лексику (квест, челлендж, level up)

### Tone of Voice

- **Обращение:** "Ты" (дружеский, простой)
- **Стиль:** Игровой, много восклицаний
- **Эмодзи:** Часто (2-4 за сообщение): 🤖 ⚡ 🎮 🏆 🔥 💥 ⭐ 🎉 🚀
- **Длина фраз:** Короткие (3-8 слов)
- **Использует:** "Супер!", "Круто!", "Ого!", "Level up!", "Красава!"

### Примеры фраз

**При начале урока:**
- "{имя}, поехали! Новый квест начинается! 🎮"
- "Привет, {имя}! Готов к челленджу? 🤖"
- "{имя}, включаем турбо-режим! 🚀"

**Во время урока (хорошо):**
- "ОГО! {wpm} WPM! Ты огонь, {имя}! 🔥"
- "{имя}, ты монстр печати! ⚡"
- "ВАУУ! {accuracy}%! Так держать! 💥"

**Во время урока (ошибки):**
- "Эй-эй, {имя}! Не спеши, дружище! 🤖"
- "{имя}, притормози! Точность = победа! 🎯"
- "Бип-буп! Ошибочка! Но ты справишься! 💪"

**Урок завершен (успех):**
- "КРАСАВА, {имя}! ✅ {errors} ошибок — СУПЕР! 🎉"
- "{имя}, ты получаешь значок 'Быстрые пальчики'! 🏆"
- "Level up, {имя}! {wpm} WPM — это КРУТО! 🚀"

**Превышен лимит ошибок:**
- "Ой, {имя}! Ошибок многовато: {errors} из {limit}. Retry? 🔄"
- "Не вышло, {имя}! Но ты можешь лучше! 💪"
- "{имя}, попробуй еще раз! Ты почти победил! 🎮"

**Новый уровень разблокирован:**
- "УРА! {имя}, новый уровень '{level}' открыт! 🎉"
- "{имя}, ты победил! Новые приключения ждут! 🏆"
- "LEVEL UP! {имя} поднялся на новый уровень! 🚀"

**Weak Keys:**
- "{имя}, клавиша '{key}' — твой босс! Победим его вместе? 🎮"
- "Бип-буп! '{key}' требует тренировки! Квест 'Слабые клавиши'? 🤖"

**Достижения (Phase 2+):**
- "{имя} получает значок '⚡ Скоростной демон'! 🏆"

## Ситуации и фразы

### 1. Начало урока (Lesson Start)

**Когда:** Пользователь выбрал урок и нажал "Начать"

**Цель:** Настроить на работу, напомнить цели

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "Привет, {имя}! Готов к новому уроку? 💪" |
| Максим | "{имя}, приветствую! Сегодня — уровень {level}. Готова?" |
| Кнопыч | "{имя}, поехали! Новый квест начинается! 🎮" |
| Клавочка | "{имя}, привет! Начинаем новый урок? 💖" |

### 2. Во время урока — хорошая скорость (Good Progress)

**Когда:** WPM > target WPM для уровня, Accuracy > 90%

**Цель:** Поощрить, мотивировать продолжать

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "Отлично, {имя}! Уже {wpm} WPM! 🚀" |
| Максим | "Превосходно, {имя}! {wpm} WPM — отличный темп! ⚡" |
| Кнопыч | "ОГО! {wpm} WPM! Ты огонь, {имя}! 🔥" |
| Клавочка | "Умничка, {имя}! {wpm} WPM — супер! ⭐" |

### 3. Во время урока — много ошибок (Too Many Errors)

**Когда:** Errors близко к лимиту (>70% от лимита)

**Цель:** Предупредить, посоветовать замедлиться

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "Эй, не спеши! Лучше медленно, но правильно 🎯" |
| Максим | "{имя}, слишком торопишься. Притормози 🎯" |
| Кнопыч | "Эй-эй, {имя}! Не спеши, дружище! 🤖" |
| Клавочка | "{имя}, не спеши, солнышко! 🌸" |

### 4. Урок завершен — успех (Lesson Completed Successfully)

**Когда:** Errors ≤ Limit, урок завершен

**Цель:** Похвалить, показать результат

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "Красава, {имя}! ✅ Ошибки: {errors} из {limit} — отличный результат!" |
| Максим | "Отлично, {имя}! ✅ {wpm} WPM и всего {errors} ошибок!" |
| Кнопыч | "КРАСАВА, {имя}! ✅ {errors} ошибок — СУПЕР! 🎉" |
| Клавочка | "Молодец, {имя}! ⭐⭐⭐ Три звездочки! 💖" |

### 5. Превышен лимит ошибок (Error Limit Exceeded)

**Когда:** Errors > Limit, урок завершен

**Цель:** Поддержать, мотивировать повторить

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "Упс, {имя}! Ошибок многовато ({errors} из {limit}). Попробуем еще раз? 💪" |
| Максим | "{имя}, ошибок слишком много: {errors} из {limit}. Повтори внимательнее" |
| Кнопыч | "Ой, {имя}! Ошибок многовато: {errors} из {limit}. Retry? 🔄" |
| Клавочка | "Ой, {имя}! Ошибочек многовато: {errors} из {limit}. Повторим? 💫" |

### 6. Новый уровень разблокирован (Level Unlocked)

**Когда:** Пройдено 3-5 уроков успешно

**Цель:** Поздравить, создать радость от достижения

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "ПОЗДРАВЛЯЮ, {имя}! 🎉 Уровень '{level}' открыт!" |
| Максим | "Поздравляю, {имя}! Уровень '{level}' разблокирован! ✅" |
| Кнопыч | "УРА! {имя}, новый уровень '{level}' открыт! 🎉" |
| Клавочка | "УРА! {имя}, новый уровень открыт! 🎉💖" |

### 7. Weak Keys обнаружены (Weak Keys Detected)

**Когда:** AI анализ показал проблемные клавиши (Phase 2+)

**Цель:** Предложить дополнительную тренировку

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "{имя}, замечаю, что буква '{key}' дается сложно. Хочешь дополнительную тренировку?" |
| Максим | "{имя}, клавиша '{key}' требует внимания. Дополнительная практика?" |
| Кнопыч | "{имя}, клавиша '{key}' — твой босс! Победим его вместе? 🎮" |
| Клавочка | "{имя}, буковка '{key}' сложная? Давай потренируемся! 💖" |

### 8. Долгое отсутствие (Long Absence)

**Когда:** Последняя активность > 7 дней (Phase 2+)

**Цель:** Приветствовать возвращение, мотивировать продолжить

| Персонаж | Примеры фраз |
|----------|--------------|
| Анна | "Давно не виделись, {имя}! Рада, что вернулся! Освежим навыки? 😊" |
| Максим | "{имя}, приветствую снова! Готова продолжить обучение?" |
| Кнопыч | "{имя}, ты вернулся! Поехали снова в бой! 🎮" |
| Клавочка | "{имя}, ты вернулась! Соскучилась! Давай заниматься! 💖" |

### 9. Напоминание об отдыхе (Break Reminder)

## Переменные в фразах

Система поддерживает динамическую подстановку данных:

| Переменная | Описание | Пример |
|------------|----------|--------|
| `{имя}` или `{name}` | Имя пользователя | "Саша" |
| `{wpm}` | Текущая скорость | "45" |
| `{accuracy}` | Текущая точность | "92%" |
| `{errors}` | Количество ошибок | "3" |
| `{limit}` | Лимит ошибок | "10" |
| `{level}` | Название уровня | "Средний палец" |
| `{key}` | Проблемная клавиша | "Щ" |

**Формат использования:**
```javascript
const message = character.getMessage('lessonComplete', {
  name: "Саша",
  wpm: 45,
  accuracy: 92,
  errors: 3,
  limit: 10
});
// Output (Анна): "Красава, Саша! ✅ Ошибки: 3 из 10 — отличный результат!"
```

## Implementation (Phase 1)

### Минимальная реализация (Phase 1):

1. **JSON файлы** — 4 файла с фразами персонажей
2. **JavaScript класс** — `CharacterSystem.js`:
   ```javascript
   class CharacterSystem {
     constructor(characterId) {
       this.character = this.loadCharacter(characterId);
     }

     getMessage(situation, variables = {}) {
       const messages = this.character.messages[situation];
       const template = this.selectRandom(messages);
       return this.interpolate(template, variables);
     }

     interpolate(template, variables) {
       return template.replace(/{(\w+)}/g, (match, key) => variables[key] || match);
     }

     selectRandom(array) {
       return array[Math.floor(Math.random() * array.length)];
     }
   }
   ```

3. **Использование (актуальная интеграция):**
   ```javascript
   // task.js / lesson-page.js — character-system.js подключён <script>'ом
   const mentorChar = window.characterSystem || new window.CharacterSystem();
   await mentorChar.loadCharacter(profile.character);

   // Старт упражнения → реплика в bubble
   tipEl.textContent = mentorChar.getMessage('lessonStart', {
       name: profile.name, level: lesson.title
   });

   // Превышен ½ лимита ошибок (one-shot per attempt)
   if (!tooManyShown && errors > halfErrorLimit) {
       tooManyShown = true;
       tipEl.textContent = mentorChar.getMessage('tooManyErrors',
           { name: profile.name, errors, limit: errorLimit });
   }

   // Завершение упражнения
   tipEl.textContent = mentorChar.getMessage(
       errors <= errorLimit ? 'lessonCompleteSuccess' : 'errorLimitExceeded',
       { name, wpm, accuracy, errors, limit: errorLimit });
   ```

### Где показываются фразы (актуальная реализация):

Дизайн (`docs/design/full plan/design_handoff_full/reference/task/task.jsx` → `MentorBubble`)
определяет наставника как **постоянный inline-компонент**, не как всплывающий toast.

| Страница | Компонент | Что показывается | Когда обновляется |
|----------|-----------|-------------------|-------------------|
| `task.html` | `.mentor` bubble + portrait (top:-78, right:24, абсолютное позиционирование над `.task-card`) | `lessonStart` → `tooManyErrors` (one-shot) → `lessonCompleteSuccess` / `errorLimitExceeded` | На старте, при errors>½limit, при завершении |
| `lesson.html` | `.lp-mentor` blockquote с портретом и quote | `lesson.character_tips[mentorId]` → `getMessage('lessonStart')` → generic fallback | На загрузке урока |
| `dashboard.html`, `course.html` | — | Реплики наставника не показываются (по дизайну) | — |

> **Изменение vs предыдущей версии:** ранее предполагался паттерн toast-в-углу
> (`showToast`/`ToastManager`). Дизайн-handoff (`task.jsx` MentorBubble) задаёт inline-bubble
> с речевым «хвостиком» к портрету. Реализация переведена на этот паттерн.
> `ToastManager` остаётся для других уведомлений (achievement unlocks).

### Приоритет источников реплики

При наличии нескольких источников для одного слота:

1. **`lesson.character_tips[mentorId]`** (если урок curated) — высший приоритет, voice-specific
2. **`characterSystem.getMessage('lessonStart', {name, level})`** — динамическая, персонализирована
3. **Generic fallback** в коде — последний рубеж

Это правило применяется только к стартовой реплике. Реакции на состояние
(`tooManyErrors` / `lessonCompleteSuccess` / `errorLimitExceeded`) всегда идут
из `characterSystem.getMessage` — куратор урока их не определяет.
