## Insights / observations

1. **Velocity на pre-sprint phase оказалась выше ожиданий**. Клод закрыл 5 топ-задач Ники + scaffold + 2 ADR за 2 дня. Это значит — когда GitHub разблокируется, мы можем держать ритм 1 sprint/неделя без перерасхода энергии. Но риск: backend solo-dev (Борис) — bottleneck в Sprint 1+; стоит подумать про парное программирование с Клодом на критичных кусках (auth, payments).

2. **API-first parallelism доказал ценность**. `verify_api_client_mock.py` (11/11 PASS) показал, что frontend api-client стабилен против реального HTTP контракта **до того**, как backend написан. Это значит — Алекс может работать параллельно с Борисом в Sprint 1-3 без блокеров, если OpenAPI публикуется на day 1-2 каждого sprint'а. Закрепим как процессное правило для всех будущих sprint'ов.

3. **Hybrid YK recurring (ADR-005)** — самое значимое решение недели. Уменьшил R-002 impact с CRITICAL до MEDIUM, но добавил 1.5d работы в Sprint 7 (S7.1 → 2.5d, S7.4 → обязательное 1.5d, новый S7.8 grace period). Sprint 7 estimate стал tight — следить за прогрессом в mid-sprint check-in.
