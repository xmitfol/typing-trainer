## 2. Ключевые тех-заметки (для будущих правок курса)
- **Печатаемость**: в `text` и step/exercise `target` — только символы своей раскладки (RU=кириллица, EN=латиница) + цифры + обычная пунктуация. НИКАКОЙ латиницы в RU; никаких «— ₽ № ~ …» / эмодзи в наборе. (Эмодзи в нарративе/character_tips — ок, не печатаются.)
- **text_sequence** — РЕГЕНЕРИРОВАТЬ из `text` по канон-карте. Скрипты-паттерны (session scratchpad): `regen_seq.py` (ЙЦУКЕН), `regen_en.py` (QWERTY), `cleanup_*.py` (спецсимволы+полный keys_trained+регенерация). finger/hand в text_sequence — НЕиспользуемые метаданные (палец берётся из клавиатуры), но держим корректным. **bracket-aware замена массивов обязательна** (наивный regex ломается на `]`/`[` в значениях).
- **keys_trained** практических уроков — полный набор (буквы + Space + цифры + пунктуация); маркер «А-Я»/«А-Я полный» = все буквы.
- **EN-практика** часто имела ПУСТОЙ text_sequence → обязательно регенерить.
- **Осторожно**: глобальная замена строк в JSON может задеть КЛЮЧИ (был баг `target_wpm`→`target_зн/мин`). Не заменять подстроки, совпадающие с ключами.
