## 🎨 Content Guidelines

### 1. Тексты по уровням:

**Уроки 1-2:** Бессмыслица OK
```
"ааа ооо ввв" - просто буквы
"ал ол ад од" - слоги
```

**Уроки 3-6:** Полу-осмысленные
```
"оно она он" - простые слова
"вода река" - существительные
```

**Уроки 7-10:** Осмысленные фразы
```
"Саша мыла раму" - простые предложения
```

### 2. Ритм-упражнения (уроки 3, 6, 9):

**Особенности:**
- `"rhythmLesson": true`
- NO new keys!
- Повторы: "ооо ооо ооо" или "оно оно оно"
- Title: "Ритм-упражнение: Закрепление"
- Description: "Тренировка ритма без новых букв"

**Tips для ритм-уроков:**
```json
"tips": [
  "Сегодня без новых букв - просто наслаждайся ритмом!",
  "Печатай ровно, в одном темпе",
  "Расслабься - это anti-burnout урок",
  "Цель: не скорость, а равномерность"
]
```

### 3. Character tips - tone of voice:

**Анна (дружелюбная наставница):**
- "Отлично! [достижение]. [совет]"
- "Хорошая работа! [что получилось]"

**Максим (профессионал с юмором):**
- "Хорошее начало! [объяснение]"
- "Молодец! [технический комментарий]"

**Кнопыч (энергичный):**
- "Вау! [восторг]!"
- "Ого! [игровой комментарий]!"

**Клавочка (милая):**
- "Какой ты молодец! [похвала]"
- "Ты умничка! [ласково]"

### 4. Progressive length:

```
Урок 1-2: 30-50 символов
Урок 3-5: 50-70 символов
Урок 6-8: 70-90 символов
Урок 9-10: 90-100 символов
```

### 5. Error limits:

```
error_limit = ceil(text_length * 0.05)

Пример:
text_length = 40 → error_limit = 2
text_length = 80 → error_limit = 4
```

## 🗺️ Finger Mapping (reference)

```javascript
fingerMapping = {
  // Left hand
  'ф': 'pinky',
  'ы': 'ring',
  'в': 'middle',
  'а': 'index',
  'п': 'index',
  'р': 'index',
  'о': 'index',

  // Right hand (mirror)
  'о': 'index',
  'л': 'middle',
  'д': 'ring',
  'ж': 'pinky',
  'е': 'index',
  'н': 'index',
  'г': 'index',

  // Space
  ' ': 'thumb'
}

handMapping = {
  // Left hand
  'ф', 'ы', 'в', 'а', 'п', 'р': 'left',

  // Right hand
  'о', 'л', 'д', 'ж', 'е', 'н', 'г': 'right',

  // Space alternates based on previous key
}
```

**Space logic:**
```
If previous key was LEFT hand → Space = RIGHT thumb
If previous key was RIGHT hand → Space = LEFT thumb
```
