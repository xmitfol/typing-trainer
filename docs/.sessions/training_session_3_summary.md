#### 1. **"Идеомоторная" печать - следующий уровень**

Книга вводит концепцию **"идеомоторной печати"** (Упр. 24):

> "Ideomotor (от англ. idea - идея + motor - движение) - в момент мысли о движении, автоматически происходит само движение без сознательного контроля."

**Это и есть ЦЕЛЬ обучения слепой печати!**

**3 уровня мастерства:**

**Level 1: Сознательный** (начинающий)
- Думает о каждой букве
- Смотрит на клавиатуру
- Медленно

**Level 2: Автоматический** (средний)
- Не смотрит на клавиатуру
- Печатает по памяти
- Средняя скорость

**Level 3: Идеомоторный** (мастер)
- Думает о СЛОВАХ, не о буквах
- Пальцы двигаются сами
- Максимальная скорость

**Для продукта:**

```javascript
// User progression tracking
user.level = {
  1: "conscious", // WPM < 30, looking at keyboard
  2: "automatic", // WPM 30-60, blind typing
  3: "ideomotor"  // WPM 60+, thinking in words
}

// Achievement unlocks
achievements = {
  "Автоматизм": "10 уроков без взгляда на клавиатуру",
  "Идеомоторика": "WPM > 60 + Accuracy > 95%",
  "Мастер": "WPM > 100 + 20 уроков completed"
}
```

**Gamification:**
- Badge: "Автопилот" (Level 2)
- Badge: "Мастер мысли" (Level 3)

#### 2. **Психологические барьеры - "стена" на уроке 24-25**

**Важное открытие:** Книга подробно описывает **психологический кризис**!

**Цитата (стр. 1037-1038):**
> "Упражнения 15-24 даются легко. Но на 24-25 упражнении многие сталкиваются с 'невидимой стеной'. Хочется бросить. Не сдавайтесь! Это проходят ВСЕ."

**Кривая прогресса:**

```
Progress
  ↑
  |         ╱╱
  |       ╱╱  (breakthrough!)
  |     ╱╱
  |    /═══ (plateau/wall)
  |   /
  |  /
  | /
  |/____________→ Lessons
   1  5  10 15 20 25 30
        ↑       ↑
     easy    WALL!
```

**Critical retention points:**
- **Lesson 3-4:** Первый мини-кризис (уже знали)
- **Lesson 7-8:** Усталость от новых букв (уже знали)
- **Lesson 24-25:** ГЛАВНЫЙ БАРЬЕР! (новое!)

**Для продукта:**

```javascript
// Barrier detection
if (lesson === 24 || lesson === 25) {
  motivationalMessage = {
    title: "Ты на правильном пути!",
    text: "Урок 24 - это 'стена', через которую проходят ВСЕ. Не сдавайся! Через 2-3 урока будет прорыв!",
    cta: "Продолжить",
    visual: "励み壁を突破しよう!" // climbing wall emoji
  }

  // Extra encouragement
  if (user.attempts[lesson] > 3) {
    message: "Ты упорный! Еще немного!"
    reward: "+50 bonus XP"
  }
}
```

**Retention tactics:**
- Email на следующий день: "Не останавливайся сейчас!"
- Push notification: "Прорыв близко!"
- In-app: Story от ученика, который прошел барьер

**KPI:**
- Lesson 24-25 completion rate (expect: lower than average)
- Dropout rate at 24-25 (monitor closely!)
- Return rate after wall (measure recovery)

#### 5. **Скорость vs Точность - вечная дилемма**

Книга дает четкий ответ (Упр. 39):

**Правило:**
> "Сначала точность (90%+), ПОТОМ скорость. Никогда наоборот!"

**Цитата:**
> "Если печатаете быстро, но с ошибками - вы учитесь ошибкам! Замедлитесь!"

**Для продукта:**

```javascript
// Speed vs Accuracy balance
if (user.wpm > 50 && user.accuracy < 90) {
  warning = {
    title: "Медленнее!",
    text: "Скорость отличная, но точность страдает. Замедлись на 10-15 WPM.",
    recommendation: "Target WPM: 40"
  }
}

// Recommended progression
targets = {
  "Lessons 1-10": { wpm: 10-20, accuracy: 90 },
  "Lessons 11-20": { wpm: 20-40, accuracy: 92 },
  "Lessons 21-30": { wpm: 40-60, accuracy: 95 },
  "Lessons 31+": { wpm: 60+, accuracy: 95+ }
}
```

**Visual feedback:**

```
If (accuracy < 90%):
  Color: Red
  Message: "Точность важнее! Замедлись"

If (accuracy 90-95% && wpm increasing):
  Color: Green
  Message: "Отличный баланс!"

If (accuracy > 95% && wpm < target):
  Color: Blue
  Message: "Можешь чуть быстрее!"
```

#### 6. **"Слепые зоны" - проблемные клавиши**

Книга описывает **типичные слабые места**:

**Топ-5 сложных клавиш (русская):**
1. **Ё** - далеко, редко
2. **Ъ, Ь** - мизинец, путаница
3. **Ж, Х** - мизинец правой руки
4. **Щ, Ш** - безымянный/мизинец
5. **Ц, У** - верхний ряд, средний палец

**Для продукта:**

```javascript
// Weak keys detection (AI)
weakKeys = user.errors.groupBy('key')
  .filter(errorRate > 20%)
  .top(5)

if (weakKeys.length > 3) {
  suggestion = {
    title: "Обнаружены слабые клавиши",
    keys: weakKeys, // ['ё', 'ъ', 'ж']
    action: "Пройти доп. тренировку?",
    drill: "weak-keys-special"
  }
}
```
