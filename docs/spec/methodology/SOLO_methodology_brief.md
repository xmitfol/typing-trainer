# Методический брифинг: СОЛО (Шахиджанян, 1992) → база для наших уроков

> **Источник:** [docs/book/solo_1992.md](../../book/solo_1992.md) — оригинал методики СОЛО (под пишущую машинку; раскладка и педагогика переносятся на клавиатуру).
> **Зачем:** наш урок 1 слишком резкий для абсолютного новичка (сразу даёт несколько букв и комбинации, минуя постановку рук). Этот разбор — основа для редизайна мягкой лестницы первых уроков + per-exercise gating.
> **Статус:** входной материал для методики (Полина) + контента (Катя) + механики (Алекс). Не финальный спек.

---

## 1. Базовые принципы методики

- **Слепой ввод — без подглядывания на клавиши.** Смотреть можно на текст и на экранную схему-подсказку, на свои руки — нет. «Подсмотрю один раз» = главный самообман.
- **Точность важнее скорости.** Скорость — отдельная поздняя фаза; сначала «медленнее, чем можешь, но точно».
- **«Нажатие — отрыв».** Лёгкое касание, после удара палец сразу возвращается на домашний ряд.
- **Расслабленные кисти** («полушатёр», пальцы — «рейки-опоры»); при напряжении — перерыв.
- **Память пальцев** — довести нахождение клавиш до автоматизма (аналогия с пианистом — сквозная).
- **Обязательный возврат на исходную позицию** — тренируется отдельно.
- **Ритм** — печатать «на счёт»; сбился с ритма → опечатки.
- **Регулярность важнее объёма** — лучше меньше, но каждый день; перерыв каждые ~30 мин.
- **Мотивация «ХОЧУ→НАДО», вера в себя** — заявленная отличительная черта курса.

## 2. Постановка рук / домашний ряд (ДО печати)

1. Левая на **ФЫВА** (Ф-мизинец, Ы-безым., В-средний, А-указ.), правая на **ОЛДЖ** (О-указ., Л-средний, Д-безым., Ж-мизинец). Большие — над пробелом.
2. **«Поставь руки вслепую»**: снять руки и не глядя попасть на исходную — требование 10 раз подряд без ошибки. Цитата: «50% успеха — правильная постановка рук».
3. Приём **«яблоко/мяч»** в ладони — запомнить ощущение свода кисти; локти ~90°.
4. Мнемоника: **ОЛДЖ = «Очень Легко Достичь Желаемого»**, АВЫФ/ОЛДЖ = «два волшебных слова».
5. «Пупочки» (тактильные точки на А/О = F/J) в книге 1992 прямо не описаны (это про физическую клаву), но идея «найти опорные клавиши вслепую» — ровно та же.

## 3. Лестница первых упражнений (КЛЮЧЕВОЕ)

**Принцип: одна клавиша — одним пальцем — много повторов, прежде чем комбинировать. От сильных пальцев (указательные) к слабым (мизинцы).**

| Упр. | Содержание |
|---|---|
| 1 | Постановка рук вслепую (ФЫВА/ОЛДЖ), 10 раз без ошибки — **без печати** |
| 2–5 | По одной букве: **А** (указ.лев) → **О** (указ.прав) → **В** (средн.лев) → **Л** (средн.прав) |
| 6–7 | Чередование освоенных `а в о л`; пары/слоги `ао ал ол во ов` |
| 8–10 | Безымянные **Ы**, **Д**; мизинцы **Ф**, **Ж** (отдельно — «капризные» слабые пальцы) |
| 11 | Слова-якоря `фыва`, `олдж`; игра «разложи 11 кружочков как на клавиатуре» |
| 12+ | **П** → первое реальное слово **«папа»**; затем Р, Э; короткие слова из освоенных букв |
| ~42 / ~67 | Верхний ряд (ЙЦУКЕ + НГШЩЗХЪ) → нижний (ЯЧСМИ + ТЬБЮЁ) — только после автоматизма предыдущего |
| 100 | Финальный панграм 10 раз без ошибок = «метод освоен» |

Нарастание: **клавиша → чередование → слог → короткое слово → длинное**; осознанное чередование «лёгкое слово — трудное» для веры в себя; новый ряд — после автоматизма.

## 4. Философия ошибок и оценок

- Переход к следующему — **только после «отлично»** (самооценка, «обманывать некого»).
- «Отлично» на одиночных = без единой ошибки (идеал «ошибся → перепечатай»), **но**:
  - объём можно сокращать (15→10 строк), есть факультативные упражнения;
  - **бюджет ошибок на фразах** (упр.38: «более 5 опечаток → перепечатать», т.е. ≤5 терпимо);
  - первые задания фактически без жёсткого учёта (автор сам это допускает).
- Ошибки **нормализуются психологически** («обычное и закономерное явление», «не ругай себя»); поощряется анализ своих типов ошибок.
- Переделка ради балла — норма и игра (на время, счёт строк без ошибки).

## 5. Мотивация и вовлечение

**Брать (механики):** «курс = игра / 100 уровней» + финальный «босс»-панграм; метафора «гаммы пианиста» (оправдывает повторы); микропобеды и измеримый прогресс (на время, серия без ошибок, **дневник занятий = прообраз стрика/статистики**); мнемоники рядов; аффирмации; чередование лёгкое/трудное.

**Отбросить (вода книги-1992):** длинные письма/дневники учеников, анекдоты, цитаты писателей, психотесты, всё про физическую машинку (лента, бензин, войлок). Тренажёр заменяет это интерактивом и мгновенным фидбэком.

## 6. Рекомендации для нашего редизайна (мягкая лестница урока 1)

1. **Начинать НЕ с печати, а с постановки рук вслепую** (шаг 0: показать дом, нащупать бугорки F/J, «поставь руки вслепую N раз» с подтверждением).
2. **Одна клавиша за раз, сильными пальцами вперёд** (указ. → средн. → безым. → мизинцы), не вываливать весь ряд сразу.
3. **Лестница «буква → чередование → слог → слово»** уже внутри урока 1; объёмы — веб-размер (20–40 нажатий, не 15 строк бумаги); к концу урока 1 — ощущение «я печатаю слова».
4. **Точность — критерий, скорость (WPM) в уроке 1 не показывать/не штрафовать.**
5. **Мягкая, не блокирующая политика ошибок** для новичка (без учёта / щедрый бюджет); строгий «без ошибок» — позже и опционально (звезда/«пятёрка»).
6. **Схема-подсказка на экране разрешена, подглядывание на свои руки — нет** (цвет-кодирование пальцев у нас уже есть → это и есть «схема»).
7. **Геймификация из «100 упражнений»:** ступени, мнемоники (детям весёлые), микропобеды/стрик, финальный панграм-ритуал блока.
8. **Единая лестница для детей и взрослых, разная обёртка** (детям — игра/картинки/короче; взрослым — «зачем»/цель + измеримый прогресс); таймер-напоминание об отдыхе.

**Главный вывод:** наш урок 1 резок из-за (а) отсутствия шага постановки рук, (б) сразу нескольких клавиш/комбинаций, (в) акцента на результат. Внедрить в первую очередь: **постановка рук вслепую как шаг 0 → по одной клавише сильными пальцами с возвратом на дом → мягкая нормализация ошибок без скорости.**
