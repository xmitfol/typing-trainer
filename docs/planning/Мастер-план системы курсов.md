**⭐ СИСТЕМА ОЦЕНКИ И ПРОГРЕССА**
**Критерии оценки уроков**
**Система 5 звезд:**
**Прогрессия по курсу**
**Разблокировка уроков:**
Линейная прогрессия (урок за уроком)
Минимум 2 звезды для разблокировки следующего
Возможность повторять любой пройденный урок
**Разблокировка блоков:**
Завершить все уроки предыдущего блока
Средняя оценка блока ≥ 3 звезды
Пройти финальный тест блока
**Статистика и аналитика**
**Отслеживаемые метрики:**
WPM (Words Per Minute) - текущий, средний, максимальный
Точность (%) - по уроку, по блоку, общая
Время практики (часы)
Слабые клавиши (где больше всего ошибок)
График прогресса (WPM по дням)
Серия успешных дней (streak)

class DifficultyAdapter {
  adjustLesson(userStats, baseLesson) {
    const performance = this.calculatePerformance(userStats);

    if (performance < 0.7) {
      // Упростить: меньше новых клавиш, больше повторений
      return this.simplifyLesson(baseLesson);
    } else if (performance > 0.95) {
      // Усложнить: добавить challenge
      return this.enhanceLesson(baseLesson);
    }

    return baseLesson; // Оставить как есть
  }
}
