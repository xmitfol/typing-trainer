## Что такое Typing Trainer?

**Typing Trainer** — это веб-приложение для обучения слепой печати (печати вслепую, не глядя на клавиатуру).

### Зачем учиться слепой печати?

- 📈 **Скорость:** печать со скоростью 60-100 WPM вместо 20-30
- ✨ **Продуктивность:** экономия часов работы каждую неделю
- 💼 **Карьера:** важный навык для программистов, писателей, офисных работников
- 🧠 **Концентрация:** вы думаете о содержании, а не о клавишах

### Что вы получите:

- 🎯 **99 Уроков** от базового до продвинутого уровня
- 🎨 **Виртуальную клавиатуру** с подсветкой клавиш
- 📊 **Real-time Stats** — мгновенная обратная связь
- ⭐ **Star Rating** — оценка ваших результатов от 1 до 5 звезд
- 💾 **Сохранение прогресса** в браузере (LocalStorage)
- 👨‍🏫 **Персонажа-наставника** — дружелюбный помощник, который будет мотивировать вас

## Ваш первый Урок: пошаговая инструкция

### Подготовка (1 минута)

1. **Сядьте правильно:**
   - Спина прямая, ноги на полу
   - Руки на клавиатуре в базовой позиции
   - Расстояние до монитора 50-70 см

2. **Базовая позиция рук:**
   ```
   Левая рука:                Правая рука:
   Мизинец  → А (Ф)          Мизинец  → ; (Ж)
   Безымянный → С (Ы)        Безымянный → Л
   Средний → В               Средний → Д
   Указательный → А (П)      Указательный → О
   Большой → Пробел          Большой → Пробел
   ```

3. **Не смотрите на клавиатуру!** (Пока можно подглядывать, но стремитесь печатать вслепую)

### Начало Урока (3 минуты)

**Шаг 1:** Выберите уровень **"Мизинец"** (розовый кружок слева)

**Шаг 2:** Нажмите **"Напечатать этот текст"** в верхней панели

**Шаг 3:** Вы увидите текст для набора, например:
```
фыва фыва фыва фыва
```

**Шаг 4:** Начинайте печатать. Приложение:
- ✅ Подсвечивает следующую клавишу (яркая подсветка на Virtual Keyboard)
- ✅ Показывает курсор (вертикальная черта)
- ✅ Окрашивает напечатанные символы:
  - **Зеленый** — правильно
  - **Красный** — ошибка

**Шаг 5:** Не спешите! Точность важнее скорости.
- Цель: **Accuracy > 90%**
- Потом увеличивайте скорость

**Шаг 6:** Завершение. Когда напечатаете все символы:
- ⏸️ Таймер остановится
- 📊 Появится финальная статистика
- ⭐ Вы получите Star Rating (1-5 звезд)

### Понимание результатов

После завершения Урока вы увидите:

```
🎉 Урок завершен!

⚡ Скорость: 28 зн/мин (WPM)
🎯 Точность: 92% (Accuracy)
⏱️ Время: 01:15
❌ Ошибки: 2 из 4 — ✅ Урок пройден!

⭐⭐⭐ 3 звезды — Хорошо! Вы на правильном пути!
```

**Расшифровка:**

| Метрика | Значение | Что это значит |
|---------|----------|----------------|
| **WPM** | 28 | Вы печатаете 28 символов в минуту (цель: 15 для начинающих) |
| **Accuracy** | 92% | 92% символов напечатаны правильно (отлично для первого раза!) |
| **Errors** | 2 из 4 | Вы сделали 2 ошибки при лимите 4 — урок пройден! |
| **Star Rating** | ⭐⭐⭐ | 3 звезды из 5 — показатель качества (для мотивации) |

**Star Rating критерии:**

| Звезды | Требования | Оценка |
|--------|------------|--------|
| ⭐⭐⭐⭐⭐ 5 звезд | WPM ≥ 100, Accuracy ≥ 95% | Превосходно! |
| ⭐⭐⭐⭐ 4 звезды | WPM ≥ 80, Accuracy ≥ 90% | Отлично! |
| ⭐⭐⭐ 3 звезды | WPM ≥ 60, Accuracy ≥ 85% | Хорошо! |
| ⭐⭐ 2 звезды | WPM ≥ 40, Accuracy ≥ 80% | Неплохо! |
| ⭐ 1 звезда | WPM ≥ 20, Accuracy ≥ 70% | Начало! |

**Важно:** Star Rating — это показатель качества для мотивации. Для прохождения урока используется **лимит ошибок** (см. ниже).

### Лимит ошибок и прохождение урока

**Критерий прохождения:** Урок считается пройденным, если вы **не превысили лимит ошибок**.

**Формула расчета лимита ошибок:**

```
Лимит ошибок = Длина текста × Error Rate

где Error Rate зависит от уровня сложности:
- Beginner (Мизинец, Безымянный):      5% (5 ошибок на 100 символов)
- Intermediate (Средний):               3% (3 ошибки на 100 символов)
- Advanced (Указательные, Большой):     2% (2 ошибки на 100 символов)
```

**Пример:**

```
Текст: 200 символов
Уровень: Мизинец (Beginner, Error Rate = 5%)
Лимит ошибок = 200 × 0.05 = 10 ошибок

Результат:
❌ Ошибки: 7 из 10 — ✅ Урок пройден! (можно переходить к следующему)
❌ Ошибки: 12 из 10 — ⛔ Превышен лимит! (нужно повторить урок)
```

**Важные правила:**

- 🎯 **Точность важнее скорости** — главное не превысить лимит ошибок
- 🔁 **Повторение при ошибках** — превысили лимит → урок нужно пройти заново
- ⭐ **Звезды для мотивации** — высокие WPM и Accuracy дают больше звезд, но не влияют на прохождение
- 📈 **Прогресс через практику** — каждое повторение улучшает технику

## Понимание статистики

### WPM (Words Per Minute)

**Что это:** Скорость печати в "словах" (по 5 символов) в минуту.

**Формула:**
```
WPM = (Всего символов / 5) / (Время в минутах)
```

**Пример:**
```
Напечатали: 150 символов
Время: 3 минуты
WPM = (150 / 5) / 3 = 30 / 3 = 10 WPM
```

**Уровни WPM:**
- 🐌 **0-30 WPM** — Начинающий (Beginner)
- 🚶 **30-60 WPM** — Средний (Intermediate)
- 🏃 **60-90 WPM** — Продвинутый (Advanced)
- 🚀 **90+ WPM** — Эксперт (Expert)

### Accuracy (Точность)

**Что это:** Процент правильно напечатанных символов.

**Формула:**
```
Accuracy = ((Всего символов - Ошибки) / Всего символов) × 100%
```

**Пример:**
```
Напечатали: 100 символов
Ошибки: 8
Accuracy = ((100 - 8) / 100) × 100% = 92%
```

**Уровни Accuracy:**
- ❌ **< 85%** — Нужно больше практики
- ⚠️ **85-90%** — Неплохо
- ✅ **90-95%** — Хорошо
- 🎯 **95-98%** — Отлично
- 🏆 **98-100%** — Идеально

**Важно:** Точность важнее скорости! Сначала достигайте Accuracy > 95%, потом увеличивайте WPM.

### Error Rate (Частота ошибок)

**Что это:** Количество ошибок на 100 символов.

**Формула:**
```
Error Rate = (Ошибки / Всего символов) × 100%
```

**Категории:**
- 🏆 **< 2%** — Превосходно
- ✅ **2-5%** — Хорошо
- ⚠️ **5-10%** — Удовлетворительно
- ❌ **> 10%** — Нужна практика


- 🟧 = Оранжевый (безымянный)
- 🟩 = Зеленый (средний)
- 🟦 = Голубой/Синий (указательный)
- 🟪 = Фиолетовый (большой)

### Подсветка клавиш (Key Highlight)

**3 типа подсветки:**

1. **Следующая клавиша** — яркая подсветка
   - Показывает какую клавишу нажимать следующей
   - Помогает найти клавишу, не глядя на физическую клавиатуру

2. **Правильное нажатие** — зеленая вспышка
   - Визуальное подтверждение правильного ввода
   - Мотивирует продолжать

3. **Ошибка** — красная вспышка (Error Flash)
   - Сигнал о неправильно нажатой клавише
   - Помогает запомнить, где вы ошиблись

### Как использовать виртуальную клавиатуру

**Для начинающих:**
1. Смотрите на виртуальную клавиатуру (не на физическую!)
2. Следите за подсветкой следующей клавиши
3. Запоминайте цвет = палец
4. Постепенно переводите взгляд на экран (текст)

**Для продвинутых:**
- Виртуальная клавиатура служит напоминанием
- Основное внимание на тексте
- Подсветка — периферийным зрением

## Советы для успеха

### 1. Правильная техника с первого дня


❌ **Неправильно:**
- Печатать 2-4 пальцами
- Смотреть на физическую клавиатуру
- Использовать "свои" пальцы для клавиш

✅ **Правильно:**
- Использовать все 10 пальцев
- Каждый палец отвечает за свои клавиши (по цвету)
- Печатать вслепую (не глядя на клавиатуру)
- Найти направляющие на клавишах А и О, для быстрого позиционирования пальцев на клавиатуру.

### 2. Точность > Скорость

**Правило:** Сначала точность, потом скорость.

```
Этап 1 (недели 1-2): Accuracy > 95% при любой скорости
Этап 2 (недели 3-4): Увеличение WPM до 30-40
Этап 3 (месяцы 2-3): WPM 60-80 при Accuracy > 95%
```

**Почему:** Неправильная техника закрепляется. Лучше печатать медленно, но правильно.

### 3. Регулярная практика

**Рекомендация:** 15-30 минут ежедневно.
Желательна тренировка каждый день. Каждый пропущенный день, это потеря практики.

```
📅 Понедельник:  3 урока × 10 минут = 30 мин
📅 Вторник:      3 урока × 10 минут = 30 мин
📅 Среда:        Выходной (отдых для мышц)
📅 Четверг:      3 урока × 10 минут = 30 мин
📅 Пятница:      3 урока × 10 минут = 30 мин
📅 Суббота:      Выходной
📅 Воскресенье:  Review Weak Keys (слабые клавиши)
```

**Важно:** Лучше 20 минут каждый день, чем 3 часа раз в неделю.

### 4. Работа над Weak Keys (слабыми клавишами)

**Что это:** Клавиши, на которых вы делаете больше всего ошибок.

**Как найти:**
- Обратите внимание на красные вспышки
- Запоминайте повторяющиеся ошибки
- В будущем (Phase 2) будет AI Weak Keys Analyzer

**Как исправить:**
- Пройдите Урок, фокусированный на этих клавишах
- Повторите 3-5 раз до Accuracy > 98%

### 5. Эргономика и здоровье

**Каждые 25 минут:**
- 🧘 Встаньте, потянитесь
- 👀 Посмотрите вдаль (снимите напряжение глаз)
- 🖐️ Сделайте упражнения для пальцев и кистей

**Правильная осанка:**
```
        👤 Голова: прямо, взгляд на экран
        |
     ──👐── Локти: угол 90°
        |
     ══╪══ Спина: прямая, опора на спинку стула
        |
       /│\ Бедра: параллельно полу
      / | \
     👞 👞 Стопы: на полу
```

### 6. Отслеживайте прогресс

**Что отслеживать:**
- 📈 WPM — должен расти каждую неделю
- 🎯 Accuracy — держите > 90%
- ⏱️ Время на урок — будет уменьшаться
- ⭐ Star Rating — стремитесь к 4-5 звездам

**Цели по неделям:**

| Неделя | WPM | Accuracy | Уровень |
|--------|-----|----------|---------|
| 1-2 | 15-20 | > 85% | Мизинец |
| 3-4 | 25-30 | > 90% | Безымянный |
| 5-8 | 35-45 | > 92% | Средний |
| 9-12 | 50-65 | > 94% | Указательные |
| 13-16 | 70-85 | > 95% | Advanced |

## Частые вопросы (FAQ)

### Где сохраняется мой прогресс?

В **LocalStorage** вашего браузера. Это означает:
- ✅ Прогресс сохраняется автоматически
- ✅ Работает без интернета
- ⚠️ Привязан к браузеру (если очистите данные браузера — потеряется)
- ⚠️ Не синхронизируется между устройствами

**Phase 2:** Будет облачная синхронизация (Backend + аккаунты).

### Сколько времени нужно, чтобы научиться?

**Зависит от целей:**

```
🎯 Базовая слепая печать (WPM 30-40):
   - 2-4 недели при 20 мин/день

🎯 Уверенная печать (WPM 60-70):
   - 2-3 месяца при 30 мин/день

🎯 Профессиональная печать (WPM 90+):
   - 6-12 месяцев регулярной практики
```

**Важно:** Все индивидуально. Не сравнивайте себя с другими.

### Можно ли пропускать уровни?

**Нет, уровни проходятся строго последовательно.**

**Почему так важно:**
- Каждый уровень развивает определенные пальцы
- Пропуск = пробелы в технике = неправильные привычки
- Правильная техника важнее быстрого результата
- Даже опытным пользователям полезно "откалибровать" технику

**Как разблокировать следующий уровень:**
1. Пройдите 3-5 Уроков текущего уровня
2. В каждом Уроке **не превысьте лимит ошибок** (см. раздел "Лимит ошибок и прохождение урока")
   - Beginner уровни: не более 5% ошибок
   - Intermediate: не более 3% ошибок
   - Advanced: не более 2% ошибок
3. Система автоматически откроет следующий уровень

**Пример прогрессии:**
```
Уровень "Мизинец" (Beginner, 5% error rate):
✅ Урок 1: ❌ Ошибки: 3 из 10 — пройден
✅ Урок 2: ❌ Ошибки: 5 из 10 — пройден
✅ Урок 3: ❌ Ошибки: 2 из 10 — пройден
→ Уровень "Безымянный" разблокирован!
```

**Для опытных пользователей:**
- Начальные уровни пройдете быстро (15-30 минут)
- Это даст правильную базу техники
- **Скоро (Phase 2-3):** Для продвинутых будут доступны:
  - 🏆 Соревнования с другими пользователями
  - 🤖 Виртуальные противники (AI с разной сложностью)
  - 🎮 Квесты и челленджи
  - 🏅 Система достижений
  - 📊 Расширенная аналитика

### Что делать, если я застрял?

**Если WPM не растет:**
- 🎯 Фокус на точность (Accuracy), скорость придет
- 🔄 Повторите предыдущий уровень
- 💪 Практикуйте Weak Keys отдельно
- 😴 Сделайте перерыв 1-2 дня

**Если Accuracy низкая:**
- 🐌 Замедлитесь! Печатайте медленнее, но правильно
- 👀 Смотрите на виртуальную клавиатуру (подсказки)
- 🔁 Повторяйте один Урок 10 раз до автоматизма
