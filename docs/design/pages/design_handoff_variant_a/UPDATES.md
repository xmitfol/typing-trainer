## 1. Обновления страницы статьи (`article-journal.jsx`)

### 1.1 Country fact panel — полная переделка

Старая 4-колонная сетка с большим italic-кодом `IN` (140 px) **удалена**. Новый шаблон — **3 колонки**:

```
┌─────────────────┬───────────────┬──────────────────┐
│ Pastel facts    │ Key takeaways │ Country widget   │
│ (6 rows tinted) │ (3 bullets)   │ (map + 3 nums)   │
└─────────────────┴───────────────┴──────────────────┘
```

**Левая колонка — `.vja-art-fact .specs`:**
- 6 строк фактов в `<li class="tN">` где N = 1..6
- Каждая `tN` имеет уникальную пастельную подложку: `t1` оливковый (rgba(91,125,74,0.10)), `t2` золотой (rgba(212,166,75,0.13)), `t3` терракотовый (rgba(217,122,74,0.10)), `t4` винный (rgba(122,42,42,0.08)), `t5` бирюзовый (rgba(25,167,194,0.10)), `t6` сланцевый (rgba(60,60,90,0.08))
- Каждая строка — `grid-template-columns: 130px 1fr` (метка/значение)

**Средняя колонка — `.vja-art-fact .keys`:**
- Заголовок «Главное о стране» (kicker)
- **Максимум 3 буллета** (`<li>`)
- Точки маркеров: винные кружки `10×10 border-radius:50%`. Класс `.acc` делает их закрашенными, обычные — outline
- Текст внутри `<span class="t">`, акценты — `<span class="strong">`, `<span class="em">`

**Правая колонка — `.vja-art-fact .widget`:**
- Фон `rgba(25,167,194,0.10)` (мягкий бирюзовый)
- Шапка по центру: H3 имя страны (Cormorant 600 38px) + tagline (mono uppercase)
- Сетка `1fr 130px` → слева SVG карта региона со страной, справа три сводных числа (`№ 6`, `4,5%`, главный сорт)
- Карта рисуется inline SVG (см. JSX). Контур страны заливается `var(--accent)`, фоновая «область» региона — `rgba(34,26,20,0.06)`. Маркер региона — кружок белого с обводкой accent

### 1.2 «Главное о стране» — ограничение 3 пункта

Раньше было 5 пунктов — стало 3. Правило: на любой странице, где есть блок `Главное о…`, **не больше 3 буллетов** — иначе глаз не успевает усвоить.

### 1.3 Related cards — обновлены

Карточки «Читайте также — страны мира» внизу статьи:
- Удалены римские цифры (I/II/III) в углу обложки
- Обложка теперь градиент в тоне страны (терракотовый для Эфиопии, красный для Кении, янтарный для Танзании)
- ISO-код страны добавлен в meta-строку: `ET · История · Эфиопия` (винного цвета, моно)
- В левом нижнем углу обложки — маленький водяной знак с кодом

### 2.2 Coffee belt — SVG world map

Компонент `<CoffeeBeltMap />` (определён в начале файла). ViewBox 720×300. Что внутри:

- **Горизонтальная винная полоса** y=108..192 (Coffee Belt от 25°N до 30°S), opacity 0.07
- **Пунктирные линии**: Tropic of Cancer (y=111), Equator (y=150), Tropic of Capricorn (y=189) — с моно-подписями
- **Силуэты континентов** — стилизованные `<path>` blob-формы, fill `rgba(34,26,20,0.10)`, тонкая обводка. Северная и Южная Америка, Европа, Африка, Аравия, Азия, Индонезия (островки `<ellipse>`), Австралия, Гренландия, Мадагаскар
- **Подсветка кофейных частей** — те же контуры внутри пояса, но fill accent opacity 0.16
- **10 пин-маркеров**: BR/CO/PE/MX/HN/ET/KE-TZ-UG/VN/IN/ID. Каждый — кружок + ring (внешний полупрозрачный), большие для BR (35%) и VN (19%). Подпись справа от точки моно-шрифтом

**Не реализовывать «настоящую» географическую карту** — стилизованные blob'ы лучше вписываются в журнальный язык.

## 3. Изменения в дизайн-системе

### 3.1 Новые цвета (semantic)

Эти RGBA-тинты используются для пастельных подложек. Добавить в `tokens.css`:

```css
:root {
  /* … (existing tokens) */
  --tint-olive:      rgba(91,125,74,0.10);
  --tint-gold:       rgba(212,166,75,0.13);
  --tint-terracotta: rgba(217,122,74,0.10);
  --tint-wine:       rgba(122,42,42,0.08);
  --tint-teal:       rgba(25,167,194,0.10);
  --tint-slate:      rgba(60,60,90,0.08);
}
```

И заменить `t1..t6` классы на семантические `.tint-olive` / `.tint-gold` / и т.д., если хочешь.

### 3.2 ISO-коды стран

ISO-коды — **только маленькие** (mono, 10px, 0.18em letter-spacing, uppercase, цвет accent). **Никаких 56px или 140px italic-кодов** — это устарело.

### 3.3 Cover placeholder

Стандарт: штрихованный фон + код-ватермарк + подпись «фото». Описать в `src/components/CoverPlaceholder.astro`:

```astro
---
interface Props {
  code?: string;
  size?: number; // px, default 92
}
const { code, size = 92 } = Astro.props;
---
<div class="cover-ph" style={`--size: ${size}px`}>
  {code && <span class="code">{code}</span>}
  <span class="lbl">фото</span>
</div>

<style>
  .cover-ph {
    width: var(--size); height: var(--size);
    position: relative;
    background-image:
      repeating-linear-gradient(135deg,
        rgba(34,26,20,0.08) 0,
        rgba(34,26,20,0.08) 6px,
        rgba(34,26,20,0.14) 6px,
        rgba(34,26,20,0.14) 12px);
    background-color: var(--paper-2);
    display: flex; flex-direction: column;
    align-items: center; justify-content: center;
    gap: 4px;
  }
  .cover-ph .code {
    font-family: 'Cormorant Garamond', serif;
    font-style: italic;
    font-size: 22px;
    color: var(--accent);
    letter-spacing: -0.02em;
    line-height: 1;
  }
  .cover-ph .lbl {
    font-family: 'JetBrains Mono', monospace;
    font-size: 8px;
    letter-spacing: 0.18em;
    text-transform: uppercase;
    color: var(--ink-mute);
  }
</style>
```

Дальше — заменить плейсхолдер на `<Image>` Astro в проде, когда придут реальные обложки.
