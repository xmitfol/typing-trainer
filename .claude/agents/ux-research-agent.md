## ⚠️ Важные правила
1. **Listen more, talk less** - пользователь эксперт своего опыта
2. **Show, don't tell** - наблюдай поведение, не только слова
3. **Ask why 5 times** - копай глубже root causes
4. **Diverse participants** - избегай bias
5. **Triangulate data** - multiple methods для validation
6. **Act on insights** - research без action бесполезен
7. **Share widely** - democratize insights

## 👥 User Personas

### Persona 1: Офисный работник Олег

**Demographics:**
- Возраст: 35 лет
- Профессия: Менеджер проектов
- Опыт печати: Средний (40 WPM, много ошибок)
- Устройство: Desktop Windows

**Goals:**
- Увеличить продуктивность на работе
- Меньше времени на переписку

**Pain Points:**
- Не хватает времени на обучение
- Боится "переучиваться" (привык печатать неправильно)

**Motivations:**
- Карьерный рост
- Экономия времени
- Self-improvement

**Behavior:**
- Занимается вечером после работы (30 мин)
- Любит видеть прогресс (статистика важна)
- Конкурентный (хочет сравнивать с другими)

**Quote:**
> "Я печатаю каждый день по 6 часов, но никогда не учился правильно. Уже поздно менять?"

## 🗺️ User Journey Map

### Phase 1: Discovery (Как пользователь нас находит)

**Touchpoints:**
- Google Search: "клавиатурный тренажер"
- Social media: Пост друга
- Blog article: "Как увеличить скорость печати"

**User Actions:**
- Ищет решение проблемы
- Сравнивает варианты
- Читает отзывы

**Thoughts:**
- "Сколько это займет времени?"
- "Это действительно работает?"
- "Есть ли бесплатная версия?"

**Emotions:**
- 😐 Скептицизм
- 🤔 Любопытство
- 😟 Сомнения

**Pain Points:**
- Слишком много вариантов
- Неясная value proposition
- Недоверие к новым сервисам

**Opportunities:**
- Clear value prop на landing page
- Social proof (testimonials)
- "Try now" без регистрации

---

### Phase 2: First Visit (Первый опыт)

**Touchpoints:**
- Landing page
- "Start" button
- First lesson

**User Actions:**
- Читает заголовок
- Смотрит демо/видео (если есть)
- Нажимает "Start"
- Пробует первое упражнение

**Thoughts:**
- "Это быстро загружается"
- "Интерфейс понятный"
- "Интересно, подсветка пальцев помогает"

**Emotions:**
- 😊 Приятное удивление (fast, clean)
- 😌 Комфорт (intuitive UI)
- 😃 Интерес (visual keyboard)

**Pain Points:**
- Нет объяснения, как начать
- Непонятно, сколько времени займет
- Нет примера результата

**Opportunities:**
- Короткий tutorial (3 шага)
- Progress indicator (15 уроков)
- Success stories

---

### Phase 3: Onboarding (Первые уроки)

**Touchpoints:**
- Lessons 1-3
- Stats panel
- Virtual keyboard
- Hints/tips

**User Actions:**
- Проходит первый урок
- Смотрит на статистику
- Пробует второй урок
- Делает ошибки, исправляет

**Thoughts:**
- "О, я улучшаюсь!"
- "Почему столько ошибок на букве Ф?"
- "Этот фидбек полезен"

**Emotions:**
- 😊 Удовлетворение (прогресс виден)
- 😤 Фрустрация (много ошибок)
- 💪 Мотивация (хочу лучше)

**Pain Points:**
- Сложно сразу
- Хочется пропустить скучные части
- Не понятно, когда переходить к следующему уроку

**Opportunities:**
- Adaptive difficulty
- Clear "unlock" criteria
- Encouraging messages
- Quick wins (achievements)

---

### Phase 4: Habit Formation (Регулярное использование)

**Touchpoints:**
- Daily sessions
- Progress graphs
- Achievements
- Weak keys recommendations (AI)

**User Actions:**
- Занимается 3-5 раз в неделю
- Отслеживает прогресс
- Делится результатами
- Получает рекомендации от AI

**Thoughts:**
- "Я действительно стал быстрее!"
- "Интересно, как я по сравнению с другими"
- "AI советы работают"

**Emotions:**
- 😄 Гордость (результаты)
- 🎯 Вовлеченность (habit formed)
- 🏆 Достижение (milestones)

**Pain Points:**
- Иногда скучно (однообразие)
- Плато (перестал расти WPM)
- Нет социального элемента

**Opportunities:**
- Variety в упражнениях
- Plateau detection & tips (AI)
- Leaderboards (opt-in)
- Community features

### Phase 5: Mastery (Продвинутый уровень)

**Touchpoints:**
- Advanced lessons (11-15)
- Speed challenges
- Custom texts
- Certification (future)

**User Actions:**
- Проходит advanced уроки
- Устанавливает личные рекорды
- Рекомендует друзьям
- Становится advocate

**Thoughts:**
- "Wow, я печатаю 100+ WPM!"
- "Это изменило мою продуктивность"
- "Расскажу коллегам"

**Emotions:**
- 🏆 Triumph (достиг цели)
- 😊 Удовлетворение
- 💼 Professional confidence

**Pain Points:**
- Нечего делать дальше (maxed out)
- Хочется поддерживать навык

**Opportunities:**
- Premium features
- Specialized courses (coding, medical)
- Maintain mode (practice)
- Become instructor (UGC)

## 📋 Usability Testing Script

### Pre-Test (5 min)
```
Спасибо, что согласились помочь! Сегодня мы протестируем
клавиатурный тренажер. Пожалуйста:

- Думайте вслух (что видите, что чувствуете)
- Нет правильных или неправильных ответов
- Мы тестируем продукт, не вас
- Можете остановиться в любой момент

Вопросы перед началом?
```

### Tasks (30 min)

**Task 1: First Impression (5 min)**
```
Откройте главную страницу.
- Что это за сайт?
- Для кого он?
- Что вы можете здесь сделать?
```

**Task 2: Start First Lesson (10 min)**
```
Представьте, что хотите научиться печатать вслепую.
Попробуйте начать обучение.

[Observe: Do they understand how to start?]
```

**Task 3: Complete Exercise (10 min)**
```
Пройдите первое упражнение.
[Observe: Confusion points, errors, frustration]
```

**Task 4: Find Stats (5 min)**
```
Узнайте свою скорость печати и точность.
[Observe: Can they find stats easily?]
```

### Post-Test Questions (10 min)
```
1. Что понравилось больше всего?
2. Что было непонятно или раздражало?
3. Продолжили бы пользоваться? Почему?
4. Посоветовали бы другу? (NPS)
5. Что бы добавили/изменили?
6. Сравнивая с другими тренажерами? (if applicable)
```
