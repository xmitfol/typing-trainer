## ⚠️ Важные правила
1. **Progressive difficulty** - от простого к сложному
2. **Engaging content** - интересные тексты, не скучные
3. **Cultural sensitivity** - учитывать культурные особенности
4. **Grammar perfect** - 0 толеранс к опечаткам
5. **JSON valid** - всегда валидировать формат
6. **Accessibility** - простой язык, понятные инструкции

## 📚 Структура курса

### Взрослый курс (15 уроков)

#### Beginner (Уроки 1-5)
- **Урок 1: Pinky** - мизинцы, домашний ряд
- **Урок 2: Pinky+** - расширение, верхний/нижний ряд
- **Урок 3: Ring** - безымянные пальцы
- **Урок 4: Ring+** - закрепление
- **Урок 5: Middle** - средние пальцы

#### Intermediate (Уроки 6-10)
- **Урок 6: Middle+** - закрепление средних
- **Урок 7: Index Left** - левый указательный
- **Урок 8: Index Left+** - расширение
- **Урок 9: Index Right** - правый указательный
- **Урок 10: Index Right+** - закрепление

#### Advanced (Уроки 11-15)
- **Урок 11: Thumbs** - большие пальцы, пробел
- **Урок 12: All Keys** - все пальцы вместе
- **Урок 13: Speed** - скоростной набор
- **Урок 14: Accuracy** - точность
- **Урок 15: Mastery** - мастерство

### Мотивационная цитата

{
  "id": 1,

### Тренировочный текст
```json
{
  "level": "pinky",
  "lesson": 1,
  "exercises": [
    {
      "id": 1,
      "title": "Домашний ряд - мизинцы",
      "instructions": "Набери текст, используя только мизинцы. Следи за положением остальных пальцев.",
      "text": "ааа ;;; ааа ;;; аа ;; а ; аа ;; ааа ;;;",
      "targetWPM": 15,
      "targetAccuracy": 90,
      "hints": [
        "Держи остальные пальцы на домашнем ряду",
        "Мизинец левой руки на букве А",
        "Мизинец правой руки на точке с запятой"
      ]
    }
  ]
}
```
