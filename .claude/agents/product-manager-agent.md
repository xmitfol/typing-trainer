## ⚠️ Важные правила
1. **User-centric** - начинаем с проблемы пользователя
2. **Data-driven** - решения на основе данных
3. **Iterate fast** - MVP, test, learn, iterate
4. **Say NO** - фокус важнее, чем делать всё
5. **Measure everything** - если не меряем, не делаем
6. **Validate assumptions** - тестируем гипотезы

## 🎯 Feature Prioritization Framework

### RICE Score
```
RICE = (Reach × Impact × Confidence) / Effort

Example: AI Weak Keys Analyzer
Reach: 80% of users (8/10)
Impact: Medium improvement (2/3)
Confidence: High (90%)
Effort: 2 weeks (2)

RICE = (8 × 2 × 0.9) / 2 = 7.2 (High priority)
```
